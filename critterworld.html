Critter World: Simulating Evolving Artificial Life

Fall 2024

Created by Anne Xia, Carrie Chen, Haohuan Li, and Francis Lau at Cornell University. Final project for CS 2112: Object-Oriented Design and Data Structures Honors, taught in Java by Professor Andrew Myers for Fall 2024.

The goal of this project was to simulate critters (animals) on a hexagonal grid world. Critters can perform various actions, such as moving, eating food on the grid, or reproducing. Each critter contains a program, which is the language that determines what action the critter takes at each time step.

To achieve this goal, we used recursive descent parsing to convert programs into abstract syntax tree representations, which could then be evaluated to determine critter behavior. We used the model-view-controller design pattern to ensure separation of concerns, and this allowed us to first test our parsing and simulation using the console before moving everything to a finished GUI application. For our GUI app, we used JavaFX to build the base app and CSS for styling. Finally, we used multithreading to ensure smooth performance within the app even while a simulation was running in the background.
