Function Dungeon

Spring 2025 | OCaml

Created by Ryan Qiu, Anne Xia, Paul Fischer, and Michaela Eichel at Cornell University. Final project for CS 3110: Data Structures and Functional Programming, taught in OCaml by Professor Anshuman Mohan for Spring 2025.

A procedurally generated dungeon-crawling game written in OCaml, a functional programming language.

My main contributions:

  • Design and implement player attack patterns and power-ups, ensuring it connects smoothly with teammates' code.
  • Implement code designs and write unit tests to verify correctness.
  • Communicate regularly with group to ensure on-time delivery of code.

Players explore randomly generated rooms, so each play-through is unique. Battle enemies of varying strengths, manage your resources, and collect power-ups to help you explore the map. Standard keyboard input is supported, and the game is displayed with custom GUI.
