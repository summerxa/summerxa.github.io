LoRA Re-implementation

GitHub | Writeup | Poster

Spring 2026 | ML, Python

Created by Anne Xia, Elliott Lin, Trisha Saini, Samarth Shridhar at Cornell University. Final project for CS 4782, taught by Professors Kilian Q. Weinberger and Wei-Chiu Ma for Spring 2026.

A re-implementation of LoRA: Low-Rank Adaptation of Large Language Models (Hu et al., 2021, arXiv:2106.09685).

The LoRA adapter reduces the number of trainable parameters needed to fine-tune a pre-trained ML model for a specific task while maintaining strong model performance. We focused on using LoRA to adapt Facebook AI's pre-trained RoBERTa-base model for sentiment classification on the Stanford Sentiment Treebank (SST-2) dataset.

My main contributions:

  • Implemented full training/validation and testing cycle with checkpointing and automatic backup to cloud storage. Implemented and debugged model class connecting RoBERTa-base model to LoRA adapter.
  • Independently developed and tested research hypotheses going beyond results stated in the paper.
  • Presented findings at course's final research panel.
  • Coordinated team discussions and to-dos to ensure on-time delivery of project milestones.
