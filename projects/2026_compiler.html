Eta Compiler

Spring 2026 | Kotlin, Compiler design

Created by Anne Xia, Farhan Naqib, Matthew Kim, and Paul Fischer at Cornell University. Final project for CS 4120: Introduction to Compilers, taught by Professor Andrew Myers for Spring 2026.

A compiler written in Kotlin for Eta, an imperative C-like programming language. Compiling an Eta program involves multiple steps: lexing, parsing, typechecking, intermediate representation (IR) generation, and finally assembly (ASM) code generation.

We also implemented the following optimizations: copy and constant propagation, constant folding, global value numbering, common subexpression elimination, tail-call optimization, function inlining, dead/redundant code elimination, register allocation, and move coalescing.
